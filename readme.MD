# Liste de parrainage

Ce projet permet de générer aléatoirement la liste de parrainage pour le WEI. C'est assez inutile et utile en même temps.
Il affiche le résultat dans un tableau la page web et exporte ce tableau en PDF.

## Installation & exécution

Téléchargez ce projet et lancez index.html

## Technologies utilisées

* JavaScript
* HTML
* Bootstrap
* JsPDF
